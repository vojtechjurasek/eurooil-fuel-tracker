# ⛽ EuroOil Fuel Tracker

Sledování aktuálních cen pohonných hmot na čerpacích stanicích EuroOil.

## Funkce

- Aktuální ceny z EuroOil Srdcovka API
- Přehled MIN / PRŮMĚR / MAX pro každý typ paliva
- Vyhledávání podle názvu, města, PSČ
- Filtrování podle typu paliva
- Řazení dle názvu, města, ceny
- Debug panel pro kontrolu API odpovědí

## Vývoj

```bash
npm install
npm run dev
```

## Deploy

Push na `main` branch automaticky spustí GitHub Actions a nasadí na GitHub Pages.
